# Smart Station
A video chat/meeting webapp using WebRTC and WebSockets (socket.io). Basically,working as  a Google Meet + a collaborative whiteboard.

## Current Features?

1. Video Calls
2. Chatbox
3. Collaborative Whiteboard
4. Screensharing

## Upcoming Features?
